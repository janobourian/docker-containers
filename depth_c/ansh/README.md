## 3. Exploring the system

```bash
ls -la
```

Options:

| Option | Long option | Description |
| :----------------: | :----------------: | :----------------: |
| -a | --all | List all files |
| -A | --almost-all | Like `-a`but without `.` or `..` |
| -d | --directory | With directory is specied |
| -F | --classify | Classify by directory or file |
| -h | --human-readable | Long format |
| -l | --long | Use a long listing format |
| -r | --reverse | Display result in reverse order |
| -S | | Sort result by file size |
| -t | | Sort by modification time |
